Bootstrap

You can load all modules of Bootstrap individually.

 In your bootsrap.scss file 
@import "~bootstrap/scss/nav"; 
@import "~bootstrap/scss/navbar";
In your bootstrap.js file 
import 'bootstrap/js/dist/collapse';
 Get started

Laravel Mix

Begin by installing Laravel Mix through NPM.

 Create a .env file 
cp .env.example .env
After creating .env file 
npm install laravel-mix --save-dev
Start the watcher 
npx mix watch
After finish run production for minified files 
npx mix --production
Read the docs

SASS

Sass will be automatically compiled and loaded into the public directory.

 Be sure you are running the newest version 
npm sass -v Current: ^1.52.2
Install newest version 
npm i sass
 Check it out

Webpack

Webpack will watch every file and copy it into the public directory.

 For creating a webp image just copy a image file into the images directory and run 
npx mix
You can find the whole configuration in 
webpack.mix.js
 Check it out

FontAwesome

Font Awesome is the Internet's icon library and toolkit, used by millions of designers, developers, and content creators.

 Import the icon you want into the fontawesome.js file 
import { faCube, faFontAwesome } from '@fortawesome/pro-thin-svg-icons'
Add the icon into the library in the fontawesome.js file 
library.add(faCube, faFontAwesome)
Add the fontawesome i-tag to any html file Find your Icon
