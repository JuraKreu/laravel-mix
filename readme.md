# Laravel Mix - Frontend Framework
#### Created by Duplexmedia

### 1. Create a .env config file
```shell script
cp .env.example .env
```

### 2. Install laravel-mix

```shell script
npm install laravel-mix --save-dev
```

### 3. Start Production

```shell script
npx mix watch
```

### Create Public directory

```shell script
npx mix
```

### Minify Production directory

```shell script
npx mix --production
```
